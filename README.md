# 🚀 Etec - Projetos e Exercícios de Lógica em C#

Este repositório reúne uma série de exercícios práticos desenvolvidos durante o primeiro módulo do curso de **Análise e Desenvolvimento de Sistemas (ADS)** na **Etec**. O objetivo é aplicar fundamentos de lógica de programação, manipulação de tipos primitivos e estruturas de controle utilizando a linguagem C#.

## 📂 Projetos Incluídos

### 1. 🧮 Calculadora Básica (Console)
Uma aplicação interativa que realiza operações aritméticas fundamentais.
- **Destaques Técnicos:** - Uso de `switch case` para seleção de operações.
  - Implementação de segurança lógica para evitar **divisão por zero**.
  - Manipulação de dados com `double.Parse`.

### 2. 🏨 Sistema de Reserva de Hotel
Algoritmo para cálculo de custos de hospedagem com lógica de negócio aplicada.
- **Destaques Técnicos:**
  - Cálculo diferenciado para crianças (meia-entrada).
  - Sistema de parcelamento de valores.
  - Formatação de saída monetária utilizando `:C` (Currency).

### 3. 📐 Cálculo de Área: Trapézio
Exercício focado em geometria analítica e precedência de operadores.
- **Destaques Técnicos:**
  - Aplicação da fórmula: $$Area = \frac{(B + b) \cdot h}{2}$$
  - Código otimizado para evitar variáveis intermediárias desnecessárias, focando em performance.

## 🛠️ Tecnologias e Conceitos Aplicados

- **Linguagem:** C# (.NET)
- **Entrada/Saída:** `Console.ReadLine()` e `Console.WriteLine()`
- **Interpolação de Strings:** Uso de `$"{variavel}"` para saídas mais limpas.
- **Tipagem:** Uso estratégico de `int` e `double` para precisão em cálculos financeiros e matemáticos.
